Scamnemesis SDK Examples

Embeddable fraud search and report widgets for any website

Search Widget

Allow users to search the Scamnemesis fraud database.

<div id="search-widget"></div>

<script>
const sdk = new Scamnemesis({ apiKey: 'YOUR_API_KEY' });
sdk.createSearchWidget('#search-widget');
</script>

Quick Verify Widget

Let users quickly check if an identifier is in the database.

<div id="verify-widget"></div>

<script>
const sdk = new Scamnemesis({ apiKey: 'YOUR_API_KEY' });
sdk.createVerifyWidget('#verify-widget');
</script>

Report Form Widget

Enable users to submit fraud reports directly from your website.

<div id="report-widget"></div>

<script>
const sdk = new Scamnemesis({ apiKey: 'YOUR_API_KEY' });
sdk.createReportWidget('#report-widget', {
    title: 'Report a Scam',
    successMessage: 'Thank you for helping protect the community!'
});
</script>

Dark Theme

The SDK supports a built-in dark theme.

<script>
const sdk = new Scamnemesis({
    apiKey: 'YOUR_API_KEY',
    theme: 'dark'
});
sdk.createSearchWidget('#dark-search-widget');
</script>

Auto-initialization (No JavaScript)

Use data attributes for zero-config setup:

<div data-scamnemesis-init="verify"
     data-api-key="YOUR_API_KEY"
     data-theme="light"
     data-language="en"></div>

<script src="https://cdn.scamnemesis.com/sdk/v1/scamnemesis.min.js"></script>
